# 🤖 Azure AI Visual Assistant

An AI-powered image analysis web application built using **Azure AI Vision** and **Google Gemini**.

## ✨ Features

* 🖼️ Image Upload
* 📝 Image Caption Generation
* 🎯 Object Detection
* 🏷️ Tag Extraction
* 🤖 Gemini AI Explanation
* 💬 Gemini Chat Assistant
* ⚡ Processing Time Tracking
* 🌙 Modern Dashboard UI

## 🛠️ Technologies Used

* Flask
* Azure AI Vision
* Google Gemini API
* HTML
* CSS
* JavaScript

## 🚀 Installation

```bash
git clone <your-repo-url>

pip install -r requirements.txt

python app.py
```

Create your own API keys and replace them in `app.py`.
